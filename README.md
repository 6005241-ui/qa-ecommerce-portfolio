# qa-ecommerce-portfolio
QA Automation Engineer portfolio — e-commerce testing,  API, UI, AI
## Цель
Стать Automation QA Engineer с фокусом на e-commerce за 14–16 недель.
## Стек технологий
- Язык: Python 
- Автоматизация: Playwright
- API: Postman, REST Client
- CI/CD: GitHub Actions
- Баг-трекинг: Jira
- AI: ChatGPT / Claude / GLM для генерации тестов и анализа
## Структура проекта
- `/manual` — ручное тестирование, тест-кейсы, баг-репорты
- `/api` — API-тестирование
- `/automation` — UI-автотесты
- `/ai` — работа с AI-инструментами
- `/reports` — аналитические отчёты
- `/notes` — конспекты
